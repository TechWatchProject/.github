### 📋 Project management

All active repos feed a single [org-wide project board](https://github.com/orgs/TechWatchProject/projects/1). Any new issue or PR auto-populates the board with `Status: Triage`.

**Fields:** `Status` · `Priority` (P0–P3) · `Issue Type` · `Date posted`
**Labels:** shared `status:` / `type:` / `priority:` / `resolution:` taxonomy + per-repo `area:` labels
**Automation:** GitHub Actions workflow in each repo (`.github/workflows/project_automation.yml`) syncs on issue/PR open/close/reopen/draft/ready-for-review
